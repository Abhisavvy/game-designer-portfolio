# Portfolio (Next.js)

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). If Reactive Resume runs on port 3000, use `npm run dev -- -p 3001`.

## Stack

- Next.js App Router, TypeScript, Tailwind CSS
- Framer Motion (optional enhancement for sections)

## Next steps

- Run `npx shadcn@latest init` if you want shadcn/ui components.
- Add case study routes under `src/app/work/[slug]/page.tsx`.
- Deploy: see [../docs/DEPLOYMENT.md](../docs/DEPLOYMENT.md).
