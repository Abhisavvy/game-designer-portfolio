## 🔧 TROUBLESHOOTING QUICK FIXES

**If navigation breaks on mobile:**
- Ensure "Menu Button" is enabled in navbar settings
- Set menu type to "Over" not "Push"

**If images look distorted:**  
- Set image "Size" to "Cover" not "Stretch"
- Enable "Responsive" image option

**If text is too small on mobile:**
- Set minimum 16px for body text  
- Use relative units (em/rem) for better scaling

**If content feels cramped:**
- Add 20px left/right padding on mobile containers
- Reduce top/bottom section padding on mobile
